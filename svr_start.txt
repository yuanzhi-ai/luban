后台发布流程配置检查
1. 检查日志路径是否配置正确 /usr/wanxiang/[svr_name]/app_data/log
2. 检查秘钥配置是否正确 包括验证码、jwt签名秘钥、手机号加密秘钥 /usr/wanxiang/[svr_name]/app_data/skey
3. 检查验证码字体是否上传 /usr/wanxiang/[svr_name]/app_data/captcha
4. 检查环境变量是否配置正确:
  MySQL
  a. WANXIANG_DB_USER
  b. WANXIANG_DB_PSWD
  c. WANXIANG_DB_IP
  d. WANXIANG_DB_PORT
  Redis
  a. WANXIANG_REDIS_ADDR
  b. WANXIANG_REDIS_PSWD

http_proxy 发布流程
1. 停止旧版的docker CONTAINER
2. 删掉旧版的 CONTAINER 和 images
3. git pull 最新版的Luban代码
4. 运行指令打docker镜像: docker build -f ProxyDockerfile -t http-proxy:latest .
5. 启动实例: docker run -dit -p 127.0.0.1:8080:8080/tcp -d -v /usr/wanxiang/http_proxy/app_data:/app/app_data -e WANXIANG_REDIS_ADDR="luban001.redis.rds.aliyuncs.com:6379" -e WANXIANG_REDIS_PSWD="Hxj@41512236" -e WANXIANG_DB_USER="wanxianghui_db" -e WANXIANG_DB_PSWD="Hxj@41512236" -e WANXIANG_DB_IP="rm-cn-lbj3bjl4l000uo.rwlb.rds.aliyuncs.com" -e WANXIANG_DB_PORT="3306" http-proxy:latest
